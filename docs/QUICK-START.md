# 快速入门指南

## 设置场景 6

在这个指南中，我们将展示如何设置场景 6。请确保您已按照以下步骤操作：

### 1. 环境准备
首先，确保您的计算机上安装了以下软件：

- Python 3.x
- pip

### 2. 克隆仓库
打开终端并运行以下命令以克隆仓库：

```bash
git clone https://github.com/martintoh96/property-image-processing.git
cd property-image-processing
```

### 3. 安装依赖
在项目目录中，运行以下命令安装所需依赖：

```bash
pip install -r requirements.txt
```

### 4. 配置环境变量
根据需要配置您的环境变量，可以在 `.env` 文件中设置。

### 5. 运行场景 6
使用以下命令启动场景 6：

```bash
python scenario6.py
```

### 6. 验证运行
确保在浏览器中打开 http://localhost:5000 ，以验证应用程序是否正常运行。

### 7. 其他帮助
如需更多帮助，请参考 [文档](https://github.com/martintoh96/property-image-processing/docs)。
